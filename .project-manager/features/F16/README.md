# Keys Page Redesign (3 Sheets)

This feature transforms the existing `Keys` view into a comprehensive AI Operations Center with three distinct sheets:

1. **API Configuration**: Central hub for API key management, with dynamic model discovery and smart auto-detection.
2. **LLM Arena**: A side-by-side text model comparison tool for prompt engineering and latency/cost benchmarking.
3. **VLM Arena**: A multimodal testing ground supporting image uploads and vision-specific parameter tuning.

This ensures prompt engineers and developers have dedicated environments for their specific testing needs, all while keeping API key management clean and secure.
